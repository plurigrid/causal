#+CINDEX: CSS

Causal CSS is a user interface for ~css-mode~. 

[[file:images/causal-css-screenshot.png]]
Causal also has support for HTML editing ([[#html-top][HTML]]).


** CSS Install
:PROPERTIES:
:CUSTOM_ID: css-install
:END:

#+CINDEX: CSS Install

In your initialization file, bind the Transient ~causal-css-tmenu~ to your key binding of preference.

#+begin_src elisp :lexical no
  (keymap-set css-mode-map "M-m" #'causal-css-tmenu)
#+end_src

A different binding ({{{kbd(M-m)}}}) is used as ~causal-css-tmenu~ is intended to work alongside with [[#editkit-install][Causal EditKit]].

** CSS Usage
#+CINDEX: CSS Usage

[[file:images/causal-css-screenshot.png]]


The following sections are offered in the menu:

- CSS :: CSS specific commands.
  - “{{{kbd(l)}}} Lookup Symbol” will reference the symbol's documentation from the Mozilla Developer Network (MDN) website.
  - “{{{kbd(c)}}} Cycle Color” should be selected when the point is on a color specification.
- Edit :: Editing commands.
  - “{{{kbd(f)}}} Indent CSS Rule” is a call to ~fill-paragraph~ whose behavior is adjusted for ~css-mode~.
- Misc :: Miscellaneous commands.
  - “{{{kbd(h)}}} Highlight Line” toggles ~hl-line-mode~ to avoid interfering the display of color specifications.
  

#+TEXINFO: @subheading CSS Settings
#+VINDEX: causal-css-settings-tmenu

The menu ~causal-css-settings-tmenu~ provides access to different ~css-mode~ settings.

[[file:images/causal-css-settings-screenshot.png]]


#+TEXINFO: @subheading CSS Unicode Symbol Support

By enabling “{{{kbd(u)}}} Use Unicode Symbols” from the Settings menu, Causal CSS will use Unicode symbols as appropriate in its menus.

For more info on using Unicode symbols, please refer to [[#ux-conventions][UX Conventions]].
