* Help
#+CINDEX: Help
#+VINDEX: causal-help-tmenu

Causal Help is a user interface for ~help-mode~, a major mode for viewing help text and navigating references in it.

[[file:images/causal-help-screenshot.png]]

** Help Install
:PROPERTIES:
:CUSTOM_ID: help-install
:END:

#+CINDEX: Help Install

In your initialization file, bind the Transient ~causal-help-tmenu~ to your key binding of preference.

#+begin_src elisp :lexical no
  (keymap-set help-mode-map "C-o" #'causal-help-tmenu)
#+end_src

~causal-help-tmenu~ deviates from the default bindings of ~help-mode-map~ as shown in the table below.

| Default Binding | Causal Binding | Command                               | Notes                                      |
|-----------------+----------------+---------------------------------------+--------------------------------------------|
| l               | M-[            | help-go-back                          | Make consistent with Causal Info behavior. |
| r               | M-]            | help-go-forward                       | Make consistent with Causal Info behavior. |
| n               | N              | help-goto-next-page                   | Use to navigate to next page.              |
| p               | P              | help-goto-previous-page               | Use to navigate to previous page.          |
|                 | n              | causal-lib-browser-forward-paragraph  | Use to navigate paragraph forward.         |
|                 | p              | causal-lib-browser-backward-paragraph | Use to navigate paragraph backward.        |

The following keybindings are recommended to support consistent behavior between ~help-mode~ and ~causal-help-tmenu~.

#+begin_src elisp :lexical no
  (keymap-set help-mode-map "M-[" #'help-go-back)
  (keymap-set help-mode-map "M-]" #'help-go-forward)
  (keymap-set help-mode-map "p" #'causal-lib-browse-backward-paragraph)
  (keymap-set help-mode-map "n" #'causal-lib-browse-forward-paragraph)
  (keymap-set help-mode-map "P" #'help-goto-previous-page)
  (keymap-set help-mode-map "N" #'help-goto-next-page)
  (keymap-set help-mode-map "j" #'forward-button)
  (keymap-set help-mode-map "k" #'backward-button)
#+end_src

** Help Usage
#+CINDEX: Help Usage

After invoking help via a ~describe-~ command, invoke ~causal-help-tmenu~ using the binding ~C-o~ (or your binding of preference).

The following sections are offered in the menu:

- Navigation :: Navigation commands with the document.
- History :: Navigate history of help invocations.
- Link :: Navigate to different references in the help buffer.
- Describe :: Get help for different Elisp types.
- Info :: If available, then open this help topic in [[file:info.org][Info]].
- Source :: Show the Elisp source. If the help displayed is for a customizable variable, then show a customize menu item.
